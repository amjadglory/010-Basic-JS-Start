# 10-Basic-JS-Start
السلام عليكم 
حابب اشارك معاكم اول تعامل ليا مع الJavaScript في دبلومه الFrontend تبع اكاديميه route
هي مجموعه تاسكات بسيطه بتخليني اتمرن على المفاهيم و الادوات البرمجيه الاساسيه زي :
1. المتغيرات (Variables)
2. أخذ مدخلات من المستخدم (Input)
3. الشروط (Conditional Statements)
4. العمليات الحسابية (Arithmetic Operations)
5. الحلقات التكرارية (Loops)
6. العمليات المقارنة (Comparison Operators)
7. الإخراج (Output)
8. حل مشاكل منطقية

و بيها حليت 20 task بسيط و هما :
1- Write a program that allows the user to enter a number then print it.
2- Write a program that takes a number from the user then prints "Yes" if that number can be divided by 3 and 4, otherwise print "No".
3- Write a program that allows the user to insert 2 integers then print the maximum number.
4- Write a program that allows the user to insert an integer then print "negative" if it's a negative number, otherwise print "positive".
5- Write a program that takes 3 integers from the user then prints the maximum and minimum elements.
6- Write a program that allows the user to insert an integer number then check if it is even or odd.
7- Write a program that takes a character from the user then prints "vowel" if it's (a, e, i, o, u), otherwise print "consonant".
8- Write a program that allows the user to insert an integer then print all numbers between 1 and that number.
9- Write a program that allows the user to insert an integer then print a multiplication table up to 12.
10- Write a program that allows the user to insert a number then print all even numbers between 1 and that number.
11- Write a program that takes two integers then prints the power (e.g., 4³ = 64).
12- Write a program to enter marks of five subjects and calculate total, average, and percentage.
13- Write a program to input month number and print number of days in that month.
14- Write a program to input marks of five subjects (Physics, Chemistry, Biology, Mathematics, and Computer), find percentage and grade.
15- Write a program to print the total number of days in a month (using switch).
16- Write a program to check whether an alphabet is vowel or consonant (using switch).
17- Write a program to find the maximum between two numbers (using switch).
18- Write a program to check whether a number is even or odd (using switch).
19- Write a program to check whether a number is positive, negative, or zero (using switch).
20- Write a program to create a simple calculator (using switch)
